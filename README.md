# Thesis

This repository contains the code and data used in my bachelor thesis investigating mesh simplification techniques and their impact on geometric accuracy.

---

## Overview

The purpose of this project is to evaluate how well different mesh simplification algorithms preserve the shape of 3D models. The evaluation is based on quantitative error metrics:

* Hausdorff Distance (maximum deviation)
* Root Mean Square (RMS) Distance (average deviation)

All experiments were performed in Blender using automated Python scripts.

---

## Contents

* `scripts/`
  Python scripts used to:

  * Import 3D models
  * Apply simplification algorithms
  * Compute Hausdorff and RMS distances
  * Log mesh statistics (vertices, edges, faces)
  * Export simplified models

* `models/`
  The 3D models used in the experiments

* `results/`
  Output data generated by the scripts (e.g., distances, statistics)

---

## Usage

1. Open Blender
2. Run the scripts from the `scripts/` folder
3. Specify the directory containing input models
4. The script will:

   * Process all models in the folder
   * Apply simplification
   * Compute error metrics
   * Save results and simplified meshes

---

## Notes

* Scripts are designed for batch processing of multiple models
* Ensure correct file paths before running
* Blender version compatibility may affect execution
